## 2c. Brief project description (what algorithms will you be comparing and on what architectures)

Algorithm 1: Bubble Sort

```
begin BubbleSort(list)

   for all elements of list
      if list[i] > list[i+1]
         swap(list[i], list[i+1])
      end if
   end for

   return list

end BubbleSort
```
